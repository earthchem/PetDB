Multiple Average function

Purpose:
This function calculates averages based on a customizable grouping column.
Calculation formulas are exported to a new worksheet.
The averages will be automatically updated once source data are modified.

This function can also be used for other aggregates, such as “STDEV”, “MIN”, “MAX” and “SUM” etc
