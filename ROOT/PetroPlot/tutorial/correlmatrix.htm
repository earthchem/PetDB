Correl Matrix function

Purpose:
This function calculates a correlation matrix for a selected data area. The output is sent to a new worksheet.

The result is similar to the correlation function in Excel Add-In “Data Analyses” tool package. The advantage of this PetroPlot function is that it outputs formulas instead of calculated values. If the source data are modified, the PetroPlot generated correlation matrix will be automatically updated.


Start:
Select a range on the worksheet.



Click PetroPlot --> Correl Matrix.



Result: (Notice that the formula instead of a value is entered in a cell.)
