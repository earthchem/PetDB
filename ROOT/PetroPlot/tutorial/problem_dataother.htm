PetroPlot Known Problems

A chart contains data from multiple files or sheets

Problem:
If an XY chart contains data from multiple files or sheets, you wouldn't be able to use PetroPlot ChangeXY or Add Data Label functions for those series.

PetroPlot only changes series with data from the current workbook. All series need to come from the SAME SHEET.

Solution:
Keep all data for an XY Chart in a single worksheet.
